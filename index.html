Junglbook

As a personal project, I put together a Facebook-inspired application that retains much of the original app's functionality such as the ability to create and edit personalized profiles, upload photos, friend other users, and more. React.js and Redux were utilized in order to keep the application front-end organized and manageable, while PostgreSQL and Ruby on Rails were used to store and serve the application data.

Using HTML5 Canvas and vanilla JavaScript, I built a simplified web browser port of the classic NES game Punchout!!! The game makes use of the JavaScript Animation Loop as well as multiple overlapping canvas layers in order to handle the game rendering and animation.
